instructions.md is the Manager. It defines the specific goal, project scope, and rules for this specific assignment.

skills.md is the Resume / Tool Belt. It defines how to perform specific reusable tasks (e.g., how to write a secure Dockerfile or how to run your project's specific test runner).

agents.md is the Org Chart. It defines who does what if you are using a multi-agent system (e.g., separating the Planner from the Coder and the Reviewer).
